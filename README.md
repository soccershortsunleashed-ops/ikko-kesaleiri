# IKKO kesäleiri — landing

Landing page for the IKKO summer camp (Punkaharju). Built with Vite + React +
TypeScript + Tailwind CSS, animations by Framer Motion.

## Highlights
- 3D coverflow photo gallery with tilt-and-zoom lightbox
- Story, reviews and farewell sections

## Development
```bash
npm install
npm run dev      # local dev server
npm run build    # production build to dist/
```

## Deployment
Hosted on Vercel with automatic deploys from this repository:
- push to `main` → production deploy
- any other branch / pull request → preview deploy

Live: https://ikko-kesaleiri.vercel.app
